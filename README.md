# 🚀 3D Mechanical Assembly Simulator

## 📌 Overview
This project is developed using Unity and provides an interactive 3D mechanical assembly experience. Users can explore different parts of a mechanical model, view individual components, and understand how they fit together to form the complete assembly.

## 🎯 Features

- Interactive 3D model visualization
- Component-wise assembly and disassembly
- Exploded view representation
- User-friendly UI controls
- Real-time part selection
- Educational learning experience for mechanical structures

## 🛠️ Technologies Used

- Unity Engine
- C#
- Unity UI System
- 3D Models
- Visual Studio

## 🎮 Controls

| Action | Description |
|----------|-------------|
| Mouse Drag | Rotate Model |
| Mouse Scroll | Zoom In/Out |
| Button Click | Show Specific Parts |
| UI Controls | Assemble/Disassemble Components |
